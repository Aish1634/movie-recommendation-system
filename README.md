# Movie Recommendation App

## Description
A web application that recommends movies based on user search using an external API. The application allows users to explore movie details through a simple and responsive interface.

## Features
- Search for movies
- Display movie details
- Responsive user interface

## Tech Stack
- HTML
- CSS
- JavaScript
- External API (TMDB)
